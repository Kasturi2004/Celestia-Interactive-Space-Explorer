# Celestia — Interactive Solar System

Celestia is a premium, fully responsive Solar System experience built with plain HTML, CSS, and JavaScript. It runs directly in a browser—no build step, framework, server, or database required.

## Project structure

```text
Celestia/
├── index.html       # Page structure and accessible UI
├── style.css        # Responsive visual system and animations
├── script.js        # Planet data, canvas stars, and interactions
└── assets/
    ├── planets/     # Reserved for future planet imagery
    └── icons/       # Reserved for future icon assets
```

## Highlights

- Animated elliptical planetary orbits, glowing planets, rotating Saturn rings, and asteroid belt
- Canvas-generated starfield, nebulae, shooting stars, occasional comets, and mouse parallax
- Planet detail dossiers with facts, descriptions, and interesting facts
- Drag-to-pan, zoom, reset, and hover distance information
- Planet comparison panel and automatic cinematic guided tour
- Realistic / colourful educational display mode
- Optional synthesized ambient space sound, enabled only after user interaction
- Keyboard-accessible controls and reduced-motion support

## Controls

| Action | How to use it |
| --- | --- |
| Explore a planet | Click or focus a planet / its orbital label |
| Close planet details | Use the close button or press `Escape` |
| Pan the system | Drag in the solar-system view |
| Zoom | Use the `+`, `−`, and reset controls |
| Compare planets | Select **Compare planets** and choose two worlds |
| Start a tour | Select **Start auto tour** |
| Change palette | Use **Color mode** in the header |
| Sound | Use **Sound off/on** in the header |
